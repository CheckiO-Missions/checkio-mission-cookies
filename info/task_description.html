This is a pretty technical mission.

You have a raw HTTP cookie. You mission is to extract value by specific name.

Input: Two arguments. Both are string. One is raw cookies, the second one is cookie-name we are looking for.

Output: String. Extracted value

Example:

{% if interpreter.slug == "js-node" %}
get_cookie('theme=light; sessionToken=abc123', 'theme') == 'light'
get_cookie('_ga=GA1.2.447610749.1465220820; _gat=1; ffo=true', 'ffo') == 'true'
{% else %}
getCookie('theme=light; sessionToken=abc123', 'theme') == 'light'
getCookie('_ga=GA1.2.447610749.1465220820; _gat=1; ffo=true', 'ffo') == 'true'
{% endif %}
